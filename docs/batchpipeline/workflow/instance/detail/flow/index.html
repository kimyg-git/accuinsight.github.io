#flow

워크플로우 목록에서 [인스턴스 목록]을 선택하면 인스턴스 작업 목록을 확인할 수 있다.

#배치 인스턴스 목록

해당 인스턴스 목록을 선택한 후 인스턴스(작업) 정보를 확인한다.

① 해당 워크플로우에서 발생한 Batch job instance 목록을 표시한다.

필드설명
클러스터해당 batch job이 실행된 cluster 정보
Job ID워크플로우가 실행될 때 생성된 Job ID
워크플로우명워크플로우 이름
시작 시각Instance가 시작된 시간
종료 시각Instance가 종료된 시간
상태상태 (Running, Suspended, Killed, Succeeded)
Action해당 Instance를 상태에 따라 제어할 수 있다. (실행, 재개, 일시정지, 종료)
각 Instance의 상태에 따라 그 시점에 사용 가능한 메뉴가 활성화 된다.

② Job Id 클릭 시 해당 Instance의 상세화면으로 이동한다.

#스케줄 인스턴스 목록

① 스케줄 실행 이력 조건 그대로 재실행 / 일시정지 / 재개 / 정지 action 버튼

② 스케줄 실행 조건 및 정보 (인스턴스 개수, 시작 시간, 종료시간, 실행 주기, 시간대, 최종수정시간)

③ 해당 Schedule에서 발생한Job instance 목록

필드설명
Cluster name해당 batch job이 실행된 cluster 정보
Instance ID워크플로우가 실행될 때 생성된 Schedule Instance ID
Nominal time해당 Job의 실행 예정시간
생성시각Instance가 생성된 시간
수정시각Instance가 수정된 시간
상태상태 (Running, Suspended, Killed, Succeeded)
Action해당 Instance를 상태에 따라 제어할 수 있다. (실행, 재개, 일시정지, 종료)
각 Instance의 상태에 따라 그 시점에 사용 가능한 메뉴가 활성화 된다.

#인스턴스 흐름

① ‘뒤로가기’ 버튼 클릭하여 인스턴스 목록으로 복귀

② 인스턴스 상세 화면에서 제공하는 정보(흐름/상세/파라미터/로그)

③ ‘흐름’탭에서 Node클릭 시 해당 Log를 제공(INFO/STD/ERR)

④ Node선택에 따라 해당 Node의 정보로 변경된다.

#인스턴스 상세

해당 Instance의 상세 정보를 제공한다.

필드설명
Job IdBatch Instance Job Id
생성된 시간Instance가 생성된 시간
시작 시간Instance가 시작된 시간
종료 시간Instance가 종료된 시간
생성자생성자
인스턴스 상태상태 (Running, Suspended, Killed, Succeeded)
DHP 콘솔해당 instance의 DHP 콘솔로 이동한다.

#인스턴스 파라미터

해당 Instance Job 실행 시 사용된 Parameter 정보를 제공한다.

#인스턴스 로그

해당 Instance Job 실행 시 발생한 Log를 제공한다.
