#데이터

딥러닝 분석에 사용할 데이터를 관리하는 영역이다. 분석대상 RAW 데이터를 [프로젝트] > [데이터]에 업로드 할 수 있으며, 데이터 관리 화면에서 데이터 전처리 상태 조회 및 타 저장소 내보내기, 삭제 작업이 가능하다.

#데이터관리

데이터 관리 화면은 생성한 프로젝트 > 데이터 선택 시 사용할 수 있다. 분석 프로젝트에서 사용하는 데이터를 관리하며 해당 메뉴를 통해 데이터 전 처리가 가능하다.

  • [데이터생성] 버튼 클릭해서 데이터 세트를 생성한다.
  • 생성한 데이터 삭제는 휴지통 버튼으로 삭제한다.
  • 생성한 데이터 조회는 데이터세트 명 설명과 생성 일자로 검색이 가능하다.
  • 데이터 세트 일괄 삭제는 삭제할 데이터 선택 후, 데이터 생성 버튼 옆에 있는 선택 항목에서 수행한다.

#데이터 세트 생성

데이터 세트 생성화면에서, 분석을 위한 RAW 데이터를 등록 할 수 있다.

  • 데이터 세트 명과 설명을 입력하고 저장 할 원천데이터의 파일명을 입력한다.
  • 데이터는 client(로컬) PC와 서버로부터 등록이 가능하다.
  • 데이터 저장 경로는 추후 custom 작업 시 데이터를 불러오는 데 활용된다.

#데이터 세트 상세

데이터를 등록 한 후, 데이터 세트 상세 화면에서 데이터의 세부 정보를 조회 할 수 있다.

  • 데이터 전처리 관리 탭에서는, 데이터 세트 내에서 원천 데이터와 전처리 된 데이터들의 리스트를 볼 수 있다.
  • 상세 탭에서는, 생성한 데이터세트의 세부 정보를 보여준다.
  • 데이터세트 상세에서, ‘데이터세트 설명’과 ‘RAW 데이터 설명’을 수정할 수 있다
  • 데이터 전처리 관리 > 데이터세트 삭제 시 원천 데이터와 전처리 결과 데이터가 모두 삭제된다.
  • 데이터 전처리 관리 > Action에서 정지 버튼으로 전처리 수행을 멈출 수 있고, 휴지통 아이콘을 클릭하면 데이터가 삭제된다.

#데이터 전처리 등록

데이터 관리> 데이터 세트 클릭 후 데이터 전처리 관리 화면에 진입 할 수 있으며, 해당 화면의 데이터 전처리 버튼을 클릭하면 데이터 전처리 등록 레이어 팝업이 나타나고 데이터 전처리를 등록할 수 있다.

  • 데이터 세트 상세 화면에서 데이터 전처리 버튼을 클릭하면 데이터 전처리 등록 화면이 나타난다.
  • 전처리 대상 데이터 선택으로 원천 데이터 또는 기존에 전처리 진행한 데이터를 선택한다.
  • 전처리 후 저장할 데이터 파일명을 입력한다.
  • 처리 데이터 설명을 작성하고 전처리 알고리즘을 선택한다.

#데이터 전처리 알고리즘

#문자벡터화

텍스트 데이터의 학습을 위해 기본 제공되는 index 파일로 문자를 벡터화하는 기능이다.

  • 입력 데이터는 각 문장이 개행으로 구분되어 있는 것을 기반으로 한다.
  • 결과는 문자 당 벡터 형태로 만들어진다.

#형태소 분석기

형태소 분석기 기능은 문장을 분석해 형태소로 분해하는 모듈로, 한글 텍스트 데이터의 학습에 사용한다

  • 분석 결과를 바탕으로 단어벡터화, word-embedding 등을 진행할 수 있다.
  • 단어 사전을 사용하려면, 단어 사전 파일(txt 파일)을 불러온 후 설정 버튼을 클릭한다. 단어 사전은 고유명사로 인식되어야 하는 단어들의 목록이다. 별도의 단어 사전이 없으면 설정 안함을 클릭한다.

#단어벡터화

단어 백터화 기능에서 텍스트 데이터의 학습을 위해 단어 벡터화를 위한 방법을 설정한다.

  • 단어 벡터화 기능을 사용할 때, Basic은 내장 된 기본 word-embedding파일을 사용해서 처리하고, custom작업에서는 직접 생성한 word embedding 파일을 로드 한다.
  • 입력 데이터는 각 문장이 개행으로 구분되어 있는 것을 기반으로 한다.
  • 결과는 단어 당 벡터 형태로 만들어진다.

#이미지 전처리

이미지 데이터의 형태, 색상, 사이즈, 회전 등을 변경할 수 있는 기능이다.

  • 이미지를 원하는 형태로 변환하는 모듈로, 이미지 변경 방법을 설정 후 저장을 클릭한다.
  • 반전, 크기 조절, 회전, 이동, 흑백 처리 등을 지원하며, 여러 변환을 동시에 적용할 수 있다.

#문장 분리

문장 분리 기능은 텍스트 학습 작업의 용이성을 위해 여러 문장을 행으로 구분한다.

  • 입력 데이터는 텍스트 파일 형태로, 개행되어 있지 않은 문장을 포함한다.
  • 300자 이하인 데이터만 문장 분리가 가능하다.

#데이터 전처리 상세

데이터 세트 상세 화면에서 생성한 전처리를 클릭 시 전처리 상세 화면으로 이동한다.

  • 전처리 데이터 설명을 변경하고, 저장을 클릭하면 데이터 설명이 변경된다.
  • 진행률은 전처리 작업이 어느정도 진행되었는지 표시한다.
  • 전처리 생성에서 입력한 정보들이 나타난다.
  • Processed 데이터 설명은 수정이 가능하다. 설명을 변경하면 저장 버튼이 활성화된다.
