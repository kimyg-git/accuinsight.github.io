DL Modeler

#DL Modeler

#소개

DL Modeler는 분산 환경 기반 딥러닝 학습을 제공하는 솔루션이다.

분석가는 DL Modeler에서 제공하는 Built-in 알고리즘을 사용하거나, Custom 작업 기능을 통해 Jupyter Notebook 에서 직접 작성한 파이썬 코드로 모델을 생성할 수 있다.

또한, 이렇게 학습한 모델을 API로 배포하여 예측할 수 있다.

#특징

  1. 규모에 관계없이 클라우드에서 딥 러닝 가속화

    • 클라우드 환경에서 프로그래밍 하거나 제공되는 Built-in 알고리즘을 활용하여 딥 러닝 모델을 구축하며, 분산 환경 하의 딥러닝 학습 및 모델 배포를 통해 보다 신속하게 설계, 개발 및 예측 서비스 제공

  2. 완전 관리형 실행 지원

    • 데이터의 가공/학습부터, 모델의 배포/예측까지 딥 러닝 분석의 Life Cycle을 완전관리형으로 제공

  3. Built-in 알고리즘을 활용한 딥 러닝 모델 구축

    • 저장된 데이터를 간편하게 탐색 및 시각화 할 수 있을 뿐 아니라, 내장 된 전처리 알고리즘 이용해 변환 후 제공되는 Built-in 딥 러닝 알고리즘 학습에 적용

  4. 단 한번의 클릭으로 모델 배포

    • 애플리케이션에 대한 인스턴스 생성, 모델 구축을 지원해 손 쉬운 모델 호스팅 및 예측 수행

#제공 기능

  1. 다양한 딥러닝 학습 방법 제공

    • Jupyter Notebook을 제공하여 직접 코드 작성 및 관리 가능하며, 제공되는 Built-in 알고리즘을 통해 클릭 몇 번 만으로 간편하게 학습

  2. 데이터 전처리/최적화된 알고리즘 제공

    • 데이터 전처리 기능을 제공하며, 데이터 특성을 파악하여 최적의 알고리즘 추천

  3. 효율적인 배포/예측 서비스 제공

    • 검증 작업을 통해 학습한 모델을 GPU 클라우드 자원에 효율적으로 분배하여 배포하고, 배포된 모델로 예측 수행

  4. 데이터/학습/배포 관리

    • 데이터 셋, 학습 내역, 배포 관리 기능을 제공하여 작업 한 내용을 쉽게 정리 가능
