Parameter

#Parameter

Parameter 탭에서 key-value 패턴으로 변수를 설정해서 사용할 수 있습니다. 워크플로우 생성화면으로 이동 후 Parameter 탭을 선택합니다. Parameter 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

  1. parameters : key-value 형태로 워크플로우에서 파라미터로 넘길 변수 입력
    • ${변수명}으로 노드별 프로퍼티에서 활용
  2. globalVariable : [설정] > [변수관리]를 통해 등록한 변수그룹 선택. parameters 보다 우선순위가 높음
  3. preserveHistory : 워크플로우 실행이력 보관주기 선택
